## 📌 Deskripsi Proyek
Website ini adalah visualisasi produk 3D interaktif bertema kopi yang menggunakan **Three.js**. Pengguna dapat berinteraksi dengan model 3D secara langsung di browser melalui teknologi WebGL.

## 🛠️ Fitur & Teknologi
- **Three.js**: Rendering model 3D (GLTF/GLB).
- **OrbitControls**: Navigasi kamera interaktif (rotasi & zoom).
- **Responsive Web Design**: Tampilan optimal di berbagai perangkat.
- **Custom Textures**: Material model 3D yang realistis.
